
# bnano

A simple, blazing fast, lightweight text editor written in C.


## Features

- Text editing
- Opens in Terminal
- Fast
- Minimal


## Installation

Build ``bnano`` with make and clang.

```bash
  git clone https://github.com/SuperbMuffin/bnano.git
  cd bnano
  make
```

Then move the binary from ``bnano/build`` to your PATH
